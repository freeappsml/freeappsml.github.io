WinMTR – Free Network Diagnostic Tool

WinMTR is a Windows application which combines the functionality of the traceroute and ping utilities in a single network diagnostic tool. Using this tool you can see where the bottleneck is, it does a traceroute then continuously pings every hop and displays statistics. This application is not updated since 2011 but its running fine under windows 10.
